Es gibt zwei Benutzer.

Benutzer 1: 
Benutzername = Test
Passwort = 123456

Benutzer 2:
Benutzername = Test2
Passwort = 123456

Von der Home Seite kann man einen neuen Benutzer erstellen oder sich anmelden. Nach dem anmelden kommt man in den Administrationsbereich, in dem man News erstellen kann. Von dort kann man zur home_admin Seite wechseln. Nach dem ändern oder löschen von News kommt man automatisch wieder zur home Seite. In der home_admin Seite kann man sich unten abmelden. In der Navbar der home_admin Seite kann man auch zur Seite passwort_ändern wechseln. News updaten oder deleten kann man in dem man bei einer News auf Detailansicht drückt. Dort findet man zwei Buttons einen zum aktualisieren und einen zum löschen.
